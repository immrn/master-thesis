* Экспорт настроек в Google Диск или в другие поставщики документов
* Импорт настроек с Google Диска или с других поставщиков документов
* Много улучшений стабильности
* Поддержка разрешений Android 6.0
Улучшенный интерфейс с материальным дизайном и поддержкой темных тем
* Панель поиска для поиска токена
* Предоставляются дополнительные сведения о токенах для лучшей совместимости с другими приложениями
* Использует современные камеры для более быстрого сканирования QR-кода
* Возможность требовать биометрическую/PIN авторизацию для запуска приложения
* Белее чем 250 встроенных значков для веб-сайтов
* Дополнительные настройки для улучшения функциональности приложения

FreeOTP - это приложение для двухфакторной авторизации (2FA) для систем, использующих протоколы одноразового пароля (OTP).
Токены можно легко добавить, отсканировав QR-код или вручную введя конфигурацию токена.

FreeOTP реализует открытые стандарты. Это означает, что никаких проприетарных серверных компонентов не требуется:
используйте любой серверный компонент, реализующий эти стандарты, например FreeIPA, который использует TOTP.
Любая реализация, соответствующая стандартам, будет работать.
В настоящее время FreeOTP предоставляет реализации HOTP (RFC 4226) и TOTP (RFC 6238).
