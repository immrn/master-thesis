FreeOTP Plus скопіював функціональність, що надає FreeOTP від RedHat з додатковими покращеннями:

* Експортування налаштувань до Google Drive або інших постачальників документів
* Імпортування налаштувань з Google Drive або інших постачальників документів
* Багато покращень стабільності
* Підтримка дозволів Android 6.0.
* Покращений КІ з підтримкою матеріального дизайну з темною темою
* Панель пошуку для пошуку ключа
* Надає більше подробиць ключа для кращої взаємодії з іншими програмами
* Використовуйте сучасне обладнання камери для швидкого сканування QR-коду
* Параметр вимагає автентифікацію через PIN / біометрику для запуску програми
* Евристичний зовнішній значок для маркерів 250+ сайтів.
* Більше параметрів для налаштування функціональності застосунку

FreeOTP це програма для двофакторної автентифікації (2FA) для систем, що використовують протоколи одноразових паролів (OTP).
Ключі можна легко додавати скануючи QR-код або вручну ввівши налаштування ключа.

FreeOTP реалізує відкриті стандарти. Це означає, що жоден власний серверний компонент не є необхідним:
використовуйте будь-який серверний компонент, який реалізує ці стандарти — наприклад FreeIPA, який використовує TOTP.
Сумісність з будь-якими стандартами реалізації.
Зараз FreeOTP забезпечує реалізацію HOTP (RFC 4226) і TOTP (RFC 6238).
